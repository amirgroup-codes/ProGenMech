# Visualization

This folder contains analysis tools for circuit activations and graph generation.

## Key files
- `compute_activations.py` — Compute top activations for latent nodes.
- `get_edge_weights.py` — Compute virtual edge weights for circuit graphs.
- `generate_pymol_view.py` — Generate PyMOL visualization scripts.
- `check_outgoing_edges.py` / `compute_outgoing_source_weights.py` — Edge analysis tools.
- `auto_discover_circuit_website.py` — Auto-discovery workflow for circuits.

## Run
- `python visualization/get_edge_weights.py --help`
- `python visualization/generate_pymol_view.py --help`
- Use the `visualization/commands.md` file as a template for common commands.

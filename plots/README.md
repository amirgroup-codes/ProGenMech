# Plot Generation

This folder contains scripts for generating visual figures from analysis results.

## Key files
- `plot_main_figures.py` — Main figure generation script.
- `plot_supp_figures.py` — Supplemental figure generation script.
- `plots/` — Additional plotting utilities and output helpers.

## Run
- `python plots/plot_main_figures.py`
- `python plots/plot_supp_figures.py`

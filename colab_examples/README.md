# Colab example data

Tiny regression dataset for testing the **Zero-shot** section of `ProGenMech.ipynb`.

- `toy_regression.csv` — first 10 variants of the GRB2 DMS assay (regression scores in `DMS_score`).

In the notebook Zero-shot section, enable **use_bundled_toy_csv** or check **upload_csv** to provide your own file.

Example sequences to score after discovery (first = wildtype; same length):

1. `TYVQALFDFDPQEDGELGFRRGDFIHVMDNSDPNWWKGACHGQTGMFPRNYVTPVNRNV`
2. `TYVQALFDFDPQEDGELGFRRGDFIEVMDNSDPNWWKGACHGQTGMFPRNYVTPVNRNV`
3. `TYVQALFDFDPQEDGELGFRRGDFIHVMDNSDPNWWKGACHGQTGMFPRNDVTPVNRNV`

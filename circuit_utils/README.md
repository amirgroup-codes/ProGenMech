# Circuit Utilities

This folder contains utilities for working with CLT and PLT circuits, including model wrappers and activation extraction helpers.

## Key files
- `clt_circuit.py` — CLT circuit helper classes and inference utilities.
- `plt_circuit.py` — PLT circuit helper classes and inference utilities.
- `progen3_activation.py` — Activation collection helpers for ProGen3 models.
- `circuit_utils.py` — Shared helpers for tokenization and circuit data.
